Tóm tắt nội dung

Trang web trình bày các thông tin về Tết Nguyên Đán: phong tục, ẩm thực, hoạt động đoàn viên và các tài nguyên đa phương tiện minh họa.

Mapping tới RUBRIC (tóm tắt)

  • Cấu trúc website: Trang chủ, trang nội dung (Nghệ thuật, Thể thao), trang đăng ký, trang báo cáo.
  • Thiết kế & CSS: Giao diện đồng bộ (màu đỏ/vàng), header/banner, footer, responsive.
  • Điều hướng: Menu chính trên mọi trang, liên kết về trang chủ.
  • Nội dung: Nội dung tập trung chủ đề Tết, có điểm nhấn.
  • Đa phương tiện: Hình ảnh trong thư mục `images/` (thêm chú thích trong trang nếu cần).
  • Trang đăng ký: `dang_ki.html` có `required`, `type` phù hợp, placeholders.
  • Tương tác: Hover cho menu/nút, focus cho liên kết.
  • Minh chứng kỹ thuật: File này + `README.md` nêu rõ cách chạy & deploy (nếu cần, dùng GitHub Pages).
  • Thiết kế mới: Áp dụng theme Tết trên toàn site.
  • Báo cáo & quy trình: Mô tả phân công (mẫu), ghi chú commit và ảnh chụp minh chứng (thêm vào repo khi có).

Hướng dẫn nhanh (Local)

  1. Mở file `index.html` bằng trình duyệt (double-click hoặc `Open With`).
  2. Để deploy: đẩy toàn bộ thư mục lên GitHub và bật GitHub Pages (branch `main`).

Gợi ý bổ sung

- Thêm ảnh Tết chất lượng (kích thước đã tối ưu) vào `images/` và kèm chú thích/nguồn.
- Thêm file `evidence/` chứa ảnh chụp màn hình và nhật ký làm việc.

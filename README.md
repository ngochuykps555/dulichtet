# TẾT NGUYÊN ĐÁN - TẾT CỦA ĐOÀN VIÊN

Hướng dẫn nhanh để chạy trang web tĩnh này:

- Mở `index.html` trong trình duyệt để xem giao diện (local).
- Để triển khai online: tạo repository trên GitHub, đẩy toàn bộ thư mục lên branch `main`, bật GitHub Pages (source: `main`/root).

Nội dung đã thực hiện:
- Trang chủ: `index.html` (giao diện Tết)
- Trang nội dung: `nghe_thuat.html`, `the_thao.html`
- Trang đăng ký: `dang_ki.html` (có trường `required`, placeholder, kiểm soát tối thiểu)
- Báo cáo & minh chứng: `report.html`

Gợi ý tiếp theo:
- Thêm ảnh Tết tối ưu (dung lượng thấp) vào thư mục `images/` và cập nhật đường dẫn.
- Thêm thư mục `evidence/` chứa ảnh chụp màn hình, nhật ký làm việc hoặc links đến commits.
